# W3School_CSS
## 语法
### 选择器{声明}<br/>
（分号分开每个声明;**每行只描述一个属性增强可读性)**<br/>
可对选择器分组，逗号分开，分享相同的声明。<br/>
子元素从父元素继承属性，单独对子元素声明则不继承父元素属性。<br/>
#### id选择器<br/>
id选择器以"#"来定义，**id属性只能在每个HTML文档中出现一次。**<br/>
格式：#name{声明}，应用：<选择器 id="name">

2.外部样式表<br/>
